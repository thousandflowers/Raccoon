# Raccoon

A Mac companion toolkit for features beyond Mole's scope.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/thousandflowers/Raccoon/main/install.sh | bash
```

## Commands

| Command  | Description                    |
|----------|--------------------------------|
| upgrade  | Update package managers       |
| ports    | Show open ports and listeners |
| battery  | Battery health and cycle count|
| backup   | Verify Time Machine status    |

## Pairs well with

[Mole](https://github.com/tw93/Mole) - Deep clean and optimize your Mac
